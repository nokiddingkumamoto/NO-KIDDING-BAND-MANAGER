NO KIDDING BAND MANAGER V7.4.0
================================

ユーザー確定デザインを基準にした、GitHub Pages用トップページ再現版です。

収録内容
- 黒背景、グランジ／飛沫／ネオン表現
- 提供された正式なNO KIDDINGロゴ
- 左メニュー、右＋ボタン
- NEXT STUDIO / NEXT LIVE
- スケジュール3件
- 3列固定のクイックメニュー
  1. 確定スケジュール
  2. スタジオ日程調整
  3. 物販・在庫
- PC／スマートフォン対応
- GitHub Pages／PWA対応

V7.3.4はフォルダを使わず、すべてのファイルをGitHub直下へ置く完全版です。
メニュー、＋、各スケジュール、クイックメニューには押せる操作領域を重ねています。

V7.4.0追加機能
- スタジオ日程調整ページ
- 回答者6名の選択
- 候補日ごとの○・△・×回答
- 回答内容の端末保存
- トップページからの画面移動

重要:
ZIP内から直接ドラッグしないでください。
Windowsの「すべて展開」を押してから、展開後の通常フォルダ内のファイルをアップロードしてください。
各メニューの本機能は、次の開発段階で追加します。
