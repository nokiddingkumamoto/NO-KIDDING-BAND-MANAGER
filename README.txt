NO KIDDING BAND MANAGER V1.0

スマホ向けPWA試作版です。

現在の機能
- スケジュール登録・削除
- メンバー出欠（○・△・×）
- 物販在庫の追加・販売・入荷
- 収入・支出と収支計算
- Android/iPhoneのホーム画面追加
- オフライン表示
- データはブラウザのlocalStorageに保存

注意
このV1.0は試作版のため、メンバー間のデータ共有やログインは未実装です。
次の段階でSupabaseを接続し、全員のスマホで同じデータを共有します。

確認方法
1. ZIPを展開
2. GitHubへアップロード
3. Vercelで公開
4. 公開URLをスマホで開く
